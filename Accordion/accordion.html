This CodePen demonstrates how we can animate the opening and closing states of an accordion with fancy reveal animation using only CSS. This concept is suitable for creating FAQ sections, Table of Contents, and more.

We cannot transition height or any CSS property from zero to auto. If we want to smoothly expand a collapsed accordion, we could set a max-height and transition the height property to a large value when we expand the section.
